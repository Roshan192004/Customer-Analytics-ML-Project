# Customer Analytics ML Project 🚀

This project focuses on solving three core business problems using the Telco Customer Churn dataset:
1. **Churn Prediction**: Classification task to predict customer attrition.
2. **Spending Prediction**: Regression task to predict customer spend.
3. **Customer Understanding**: Clustering task to group similar customers.

## Project Phases
- **Phase 1**: Problem Understanding & Dataset Setup (Completed ✅)
- **Phase 2**: Data Cleaning & Preprocessing (Next 🔜)
- **Phase 3**: Exploratory Data Analysis (EDA)
- **Phase 4**: Feature Engineering
- **Phase 5**: Model Building & Evaluation

## Dataset
The dataset used is the [Telco Customer Churn dataset](https://raw.githubusercontent.com/treselle-systems/customer_churn_analysis/master/WA_Fn-UseC_-Telco-Customer-Churn.csv).
